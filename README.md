# 🚀 funenglishkids.github.io

Trang tổng hợp các ứng dụng web của **Nguyễn Ngọc Bình** – các công cụ học tiếng Anh miễn phí cho trẻ em & tiện ích vui vẻ.

🔗 **Live:** [https://funenglishkids.github.io](https://funenglishkids.github.io)

## 🌟 Các ứng dụng

| # | Ứng dụng | Mô tả | Link |
|---|----------|-------|------|
| 🎮 | **Play & Learn English** | Học tiếng Anh qua 15 trò chơi: Flashcards, Quiz, Hangman, Bingo, Memory, Word Search, Spelling… | [Truy cập](https://funenglishkids.github.io/PlayAndLearnEnglish) |
| 🗣️ | **Speak English Kids** | Luyện nói tiếng Anh cho trẻ 7-10 tuổi: Đọc to, Lặp lại, Hỏi đáp, Nhập vai, Nói nhanh, Mô tả tranh | [Truy cập](https://funenglishkids.github.io/SpeakEnglishKids) |
| 📝 | **Grammar English Kids** | Luyện ngữ pháp: 12 chủ đề, 6 loại bài tập, hướng dẫn học, từ điển tra cứu, lộ trình học | [Truy cập](https://funenglishkids.github.io/GrammarEnglishKids) |
| 🐦 | **Flappy English** | Học từ vựng nâng cao qua game Flappy Bird: 200+ từ IELTS/TOEFL, luyện nghe, Speed Quiz, Dictation | [Truy cập](https://funenglishkids.github.io/FlappyEnglish) |
| 🔮 | **Tử Vi Vui Vẻ** | Xem lá số Tử Vi online: Lập lá số 12 cung, luận giải, vận hạn, tương hợp, tra cứu sao | [Truy cập](https://funenglishkids.github.io/tuvi) |

## 🛠️ Công nghệ

- **HTML5** – Semantic markup
- **CSS3** – Grid, Flexbox, animations
- **Vanilla JavaScript** – Không framework
- **Web Speech API** – Phát âm & nhận diện giọng nói
- **GitHub Pages** – Hosting miễn phí
- **Responsive Design** – Tương thích mọi thiết bị

## 🚀 Getting Started

1. Clone repo:
   ```bash
   git clone https://github.com/funenglishkids/funenglishkids.github.io.git
   ```
2. Mở `index.html` trong trình duyệt
3. Không cần build, không cần server!
